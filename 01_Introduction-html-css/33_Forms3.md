### Overview
The **Form styling tutorials** section comes after learning form structure and controls. It teaches how to style forms effectively with CSS. 

Forms are notoriously difficult to style because browsers apply default platform-specific styles. These tutorials cover basic to advanced techniques, including modern ways to fully customize tricky elements like `<select>`.

This section is **intermediate** level and assumes you already know basic HTML forms and CSS.

### Key Takeaways
- Styling forms requires a mix of **CSS** and sometimes **JavaScript** for full customization.
- Browser defaults vary significantly across platforms (desktop vs mobile, different OS).
- Modern CSS features (especially for `<select>`) give much better control than older methods.
- **UI pseudo-classes** are extremely powerful for creating interactive, accessible form feedback.
- Always test styling across multiple browsers and devices.

### Recommended Learning Order (Form Styling Phase)

3. **Customizable select elements** + **Customizable select listboxes** → Learn modern select customization.
4. **UI pseudo-classes** → Add state-based styling.
